Generación de polígono de campo con Machine Learning

Publicado : July 09, 2022

El objetivo de este proyecto es generar polígonos con la forma de las chacras en base a los datos de gps de las cosechadoras o sembradores que trabajen en ellos.

Si bien los datos de gps indican una serie de puntos por los que la máquina pasó, no hay una forma programática de determinar la forma correcta de las chacras, ya que son muy irregulares y puden haber pérdidas de datos, por lo que se entrenó un modelo de Machine Learning con la librería FastAI que, en base a estos puntos, determine la forma de la chacra y la retorne como polígono.

Lo primero que se hace es generar, en base a chacras con la forma ya conocida, una serie de imagenes con los puntos de gps que esten dentro. El formato que admite FastAI para los problemas de segmentación como este es: una imagen PNG desde la que se va a predecir, y una máscara binaria (tambien en png) que indica que puntos pertenecen a la chacra y cuales no.

Se entrena un modelo de segmentación que retorna una matriz de probabilidades de que un punto pertenezca a la chacra en base a una imagen con los puntos.

Sabiendo las coordenadas de las esquinas de las imágenes generadas para la predicción, se puede convertir la matriz obtenida de vuelta en un polígono, descartando los puntos con baja probabilidad de pertenecer a la chacra. Para esto se usa también la librería Imantics, que tiene algunas utilidades para esta conversión.

Finalmente, obtenemos un resultado como un poligono de shapely, en la imagen anterior tenemos:

  1. A la izquierda: LineString del recorrido del vehículo
  2. Al centro: Puntos individuales del LineString (esto se utiliza como input para el modelo)
  3. A la derecha: Polígono resultante obtenido por el modelo.

Una utilidad inesperada, es que el modelo generaliza de forma que incluso en chacras de las que se tienen pocos datos se obtiene un polígono que se ajusta suficientemente bien a la forma correcta.
